# java-theater-management-system
Project Description

This project is a simple Theater Management System developed using Java.
It allows users to view movies, book tickets, and manage theater information using a console-based application.

Features

Display available movies

Book movie tickets

Show theater details

Simple console output system

Technologies Used

Java

Object Oriented Programming (OOP)

Project Structure

│
└── README.md

How to Run

Download or clone the project

Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)

Run the Main.java file
